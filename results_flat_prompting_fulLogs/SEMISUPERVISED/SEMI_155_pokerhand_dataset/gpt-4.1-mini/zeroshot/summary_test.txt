## Insights / Lessons Learned:

- **Data Compatibility:** The pipeline step expecting sparse input failed on dense DataFrame data; pre-processing type conversions need attention.
- **Scalability Limits:** Kernel-based semi-supervised methods like label propagation face scalability issues (large affinity matrix memory explosion) on big data.
- **AutoML Strategy:** Sampling down is crucial but might not fully prevent heavy resource consumption of some algorithms.
- **Importance of Error Handling:** Logging detailed errors helps diagnose failure reasons in pipeline synthesis and evaluation.

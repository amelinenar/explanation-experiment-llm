 AutoML Process Summarization

### Data Profiling Phase
- Columns: 2
- Column 0 ('Date'): Text type, identified as DateTime.
- Column 1 ('Close'): Float type.
- Non-numeric features detected: DATETIME_ENCODER.
- No missing values.
- Dataset sampled down from 3712 to 2001 rows.

### Grammar and Task Setup
- Task: TIME_SERIES_FORECAST_TASK
- Grammar includes pipeline elements: IMPUTER, ENCODERS, FEATURE_SCALER, FEATURE_SELECTOR, REGRESSOR, and TIME_SERIES_FORECAST.
- Several options for each component are listed, such as multiple encoders (OrdinalEncoder, CyclicalFeature, etc.), scalers, selectors, regressors, and forecasting estimators (ArimaEstimator, DeeparEstimator, NBEATSEstimator, NHITSEstimator).

### Pipeline Search and Synthesis Phase

#### Iteration 1 - Monte Carlo Tree Search (MCTS) Simulations

- Multiple simulations performed exploring pipeline structures:
  - Initial pipeline structure attempts involved FULL pipelines with IMPUTER, ENCODERS, FEATURE_SCALER, FEATURE_SELECTOR, and REGRESSOR.
  - Alternate path with IMPUTER and TIME_SERIES_FORECAST components.

- Move actions expanded pipeline components stepwise:
  - ENCODERS assigned to DATETIME_ENCODER.
  - IMPUTER set to 'E' (empty).
  - FEATURE_SCALER chosen as sklearn.preprocessing.MaxAbsScaler or RobustScaler in different trials.
  - FEATURE_SELECTOR tested with GenericUnivariateSelect, SelectPercentile, SelectKBest, or omitted.
  - REGRESSOR tested various regressors including ARDRegression and Ridge.

- Pipeline builds included constructing ColumnTransformer with OrdinalEncoder on the time column.

- Some pipelines scored with exceptions during cross-validation:
  - Errors raised due to zero features after feature selection steps for ARDRegression and Ridge models.
  
- Successful pipelines were created after adjustments:
  - Example of a built pipeline: 
    ColumnTransformer (OrdinalEncoder on Date) → RobustScaler → GenericUnivariateSelect → Ridge regressor.
  - Achieved scores reported, e.g., 40.97 and improved to 61.21.

- Frequent transitions involved enabling/disabling FEATURE_SELECTOR to address zero-feature issues.

- TIME_SERIES_FORECAST pipelines explored using different estimators like NHITSEstimator, DeeparEstimator, ArimaEstimator, and NBEATSEstimator.

- The best pipeline found used robust scaling, ordinal encoding of the date, feature selection, and Ridge regression, achieving a top score of 61.2188.

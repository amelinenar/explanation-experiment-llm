   - **FEATURE_SCALER**: Scales features to a standard range, often necessary for algorithms sensitive to scale (e.g., SVM, k-NN).
   - **FEATURE_SELECTOR**: Selects important features to enhance model performance and reduce overfitting.
   - **REGRESSOR**: The predictive algorithm used to make forecasts.
